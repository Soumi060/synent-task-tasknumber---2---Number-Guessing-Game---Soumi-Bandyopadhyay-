# 🎮 Number Guessing Game (CLI)

A simple Python command-line Number Guessing Game for beginners.

## 📌 Objective

The program generates a random number, and the player has to guess it correctly with the help of hints.

---

## ✨ Features

* Random number generation
* Unlimited guessing attempts
* Hint system:

  * Too High
  * Too Low
* Attempt counter
* Handles invalid inputs

---

## 🛠️ Technologies Used

* Python 3
